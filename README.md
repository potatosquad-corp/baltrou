# How to run
1. Download nodejs

Go check their download page, also install npm and pnpm

2. install packages

execute the following command
```bash
pnpm install --prod
```
3. Configure the env file
 
Edit the `.env` file
for the twitch variables you can find them in the developper portal.

4. Start the app
```bash
npm run start
```
